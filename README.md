# WeatherMate
## A real-time meteorological dashboard providing location-based weather data with search feature and Air Quality Index (AQI) metrics.

## Overview
Weather Web App is a lightweight, responsive application designed to provide users with accurate, real-time weather conditions. By leveraging the browser's Geolocation API and third-party weather data providers, the application delivers instant meteorological data for the user's current location, as well as global search capabilities.

Beyond standard temperature and precipitation data, this application integrates AQI (Air Quality Index) monitoring, making it an essential tool for planning outdoor activities such as running, cycling, or commuting.

## Key Features:
Automatic Geolocation: Utilizes the HTML5 Geolocation API to automatically detect coordinates and fetch hyper-local weather data upon initialization.
Global Search Functionality: Robust search mechanism allowing users to query weather conditions for any city or region worldwide.
Air Quality Integration: Real-time AQI data visualization to assist users in monitoring environmental health standards.
Responsive Design: Built with a mobile-first approach, ensuring a seamless experience across desktops, tablets, and mobile devices.
Minimalist UI: Clean, intuitive interface focusing on data readability and user experience (UX).

## Tech Stack
**Frontend**: HTML5, CSS3, JavaScript (ES6+)

**Architecture**: Client-side rendering (SPA feel)

## APIs:
- HTML5 Geolocation API
- WeatherMap API (Weather Data)
- WeatherMap API (AQI Data)

## Important Configuration
Geolocation Permissions: For the automatic location detection feature to function, you must allow location access when prompted by your browser. Ensure your device's location services are turned on.
